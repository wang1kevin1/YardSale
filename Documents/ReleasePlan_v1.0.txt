USER STORIES:
Sprint 1:
- As a user, I want to be able to access the product from my android device via a download. (1)
- As a user, I want a basic data structure on the server that will be able to hold user data, 
	even though I won't be able to necessarily access it directly. (13)
- As a user, I want to be able create an account that is SIM-specific/password/fingerprint protected. (8)
- As a user, I want to see a search tool for me to enter keywords on the "Browsing" page. (3)
- As a user, I want to see a form where I can fill in a descriptive title and details -- including 
	location -- to create my post. (5)
- As a user, I want to be able to attach pictures to my post. (5)
- As a user, I want to go to the create-a-post form when I select a button on the "My Posts" page. (5)

Sprint 2:
- As a user, I want a basic database on the server that will be able to hold a created post's data. (13)
- As a user, I want to be able to view my current posts on the "My Posts" page. (8)
- As a user, I want to be able to see the summarized posts (picture and title) that include my input 
	keywords on the "Browsing" page after entering a search. (8)
- As a user, I want to be able to select the summary tile and open a detailed profile of the post. (8)
- As a user, I want my posts to self-delete after two weeks. (5)
- As a user, I want to see a widget to switch between "Browsing", "My Posts" and "My Account Settings". (5)

Sprint 3:
- As a user, I want to be able to define the distance parameter of my search by location. (5)
- As a user, I want to see only posts within my range if I have defined a distance parameter in my search. (13)
- As a user, I want to be able to delete my current posts on the "My Posts" page. (5)
- As a user, I want to be able to change my password in "My Account Settings". (5)
- As a user, I want a forgot my password feature on "Login". (5)
